# 🚀 Social Media API (Node.js + Express + MongoDB)

Backend API :
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- MVC Architecture

---

## ✨ Features

- 👤 **User CRUD**
- 📝 **Post CRUD**
- 🔑 **Login & Signup**
- 🔒 **Authentication & Role Restriction (admin, user)**
- 🛠 **Create First Admin**
- 🖼 **Upload Image for Post**
- 📂 **MVC Project Structure**
- 📌 **Rate Limiting
